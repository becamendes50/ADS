Visualização de Dados para Análise de Comportamento de Algoritmos Estocásticos
×

Informações relativas ao projeto

O objetivo deste projeto passava por implementar uma ferramenta para visualização de dados que permitisse a análise comportamental de algoritmos estocásticos. Esta ferramenta permite que sejam visualizados dados sem limite máximo de variáveis e incorporando dados de vários ficheiros com o mesmo número de variáveis.

Este projeto foi realizado pelos alunos abaixo mencionados, no âmbito da cadeira de Arquitetura e Desenho de Software [ADS], lecionada pelo professor Vítor Basto Fernandes e pertencente ao plano curricular do 1º ano do Mestrado em Engenharia Informática [MEI] do ISCTE - Instituto Universitário de Lisboa.
